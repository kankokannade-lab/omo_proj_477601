Проект: Интеллектуальный оценщик сложности энкаунтеров в D&D 5e

Реализация учебного проекта по дисциплине «Основы машинного обучения». 
Проект представляет собой ИИ-сервис, который оценивает относительную угрозу монстра для конкретной боевой партии игроков.

Команда проекта:
Студент 1 (Фомин Егор Александрович): Ветка k-Nearest Neighbors (k-NN), масштабирование данных, подбор гиперпараметров.
Студент 2 (Матяс Полина Дмитриевна): Ветка Decision Tree (Дерево Решений), визуализация структуры дерева, подбор гиперпараметров.

Результаты сравнения моделей:
В ходе работы были обучены и настроены через `GridSearchCV` два алгоритма классификации:
Decision Tree Accuracy: 86%
k-NN Accuracy: 81%

Вывод: 
Модель Дерева Решений показала более высокую точность (86%) и была выбрана в качестве финального "ядра" для веб-интерфейса. Дерево решений лучше интерпретирует игровой баланс и не требует предварительного масштабирования признаков при инференсе.

Установка зависимостей:
Убедитесь, что у вас установлен Python 3.10+. Откройте терминал в папке проекта и выполните:

pip install -r requirements.txt
